Help getting started using discord bots

    Setting up your own bot


  1. Go to Discord Developer Tool.

  2. Then go to Applications on the side menu.

  3. Click on "New Application" and give it a name.

  4. Then on the settings menu click on "Bot" and "Add Bot".

  5. After you created your Bot, go over to "OAuth2" and scroll down to "OAuth2 URL Generator"
    then mark "Bot" and "applications.commands"(for slash commands).

  6. Once you've done that, you will get a "Bot Permissions" menu under, then mark "Administrator".

  7. Then copy the URL and paste it in your browser and select your discord server.

  8. God job! Now you have your own bot that you can do almost what ever you like it to do.

    9. REMEMBER! NEVER SHARE YOUR TOKEN WITH ANYONE


    Use my pre-build bots


  1. Click on "Get Started" on the bot you like to add to your server.

  2. When you're using the AuditLog bot you have to type "-help" if you dont know how to use it.

  3. When you're using the standard Music Bot you have to type ">help" if you dont know how to use it.
    You can also change the command prefix ">" with this bot.

  4. If you have problems setting up the bots, go over to my discord support server and we'll fix your bot, so it will work!
