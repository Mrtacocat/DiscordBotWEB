COMMANDS

Commands for each bot.

    Music Bot - Prefix [>] (changeable)


  1. Bassboost [None | Low | Medium | High] - Enables bass boosting audio effect - Aliases [bb, bass]

  2. Bump - Moves a track to the front of the queue - Aliases [b]

  3. Clear - Clears the server queue - Aliases [cl, cls]

  4. Config - Edit the bot settings - Aliases [conf]

  5. Disconnect - Stop the music and leave the voice channel - Aliases [leave, exit, quit, dc, stop]

  6. Grab - Saves the current song to your Direct Messages - Aliases [save]

  7. Help [command] - Information about the bot - Aliases [command, commands, cmd]

  8. Invite - To invite me to your server - Aliases [inv]

  9. Loop - Loop the current song - Aliases [l, repeat]

  10. Loopqueue - Loop the whole queue - Aliases [lq, repeatqueue, rq]

  11. Lyrics [Song Name] - Shows the lyrics of the song searched - Aliases [ly]

  12. Move - Moves a track to a specified position. - Aliases [m]

  13. Nowplaying - See what song is currently playing - Aliases [np, nowplaying, now playing]

  14. Pause - Pauses the music

  15. Play [song] - Play your favorite songs - Aliases [p]

  16. Queue - Shows all currently enqueued songs - Aliases [q]

  17. Remove [number] - Remove a song from the queue - Aliases [rm]

  18. Resume - Resumes the music

  19. Search [song] - Shows a result of songs based on the search query - Aliases [se]

  20. Seek [time s/m/h] - Seek to a position in the song - Aliases [forward]

  21. Shuffle - Shuffles the queue - Aliases [shuff]

  22. Skip - Skip the current song - Aliases [s, next]

  23. Skipto [number] - Skip to a song in the queue - Aliases [st]

  24. Stats - Get information about the bot - Aliases [about, ping, info]

  25. Volume [volume] - Check or change the current volume - Aliases [vol, v]

  26. Youtube - Starts a YouTube Together session - Aliases [yt]

    27. Got any questions? Head over to my Discord Support Server.


    AuditLog Bot - Prefix [-]


  1. Enable [number] - enable the logging for a module

  2. Enable all - enable the logging for a module

  3. Disable [number] - disable a logging module

  4. Disable all - disable all logging modules

  5. Reset - refreshes the bots entire cache for the server; everything set to default, with no logging channel
